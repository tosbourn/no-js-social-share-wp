== Description ==
This plugin adds sharing functionality to your pages that doesn\'t require JavaScript.

Currently we support;

* Twitter
* Facebook
* Google Plus
* LinkedIn
* Buffer
* Reddit
* Hacker News

== Installation ==
1. Upload \\\"no-js-social-share.php\\\" to the \\\"/wp-content/plugins/\\\" directory.
1. Activate the plugin through the \\\"Plugins\\\" menu in WordPress.
